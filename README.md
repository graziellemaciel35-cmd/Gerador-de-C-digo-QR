# Gerador-de-C-digo-QR
Projeto prático e responsivo desenvolvido com HTML, CSS e JavaScript, que permite criar códigos QR a partir de textos, mensagens ou URLs
 O layout é organizado em duas colunas: formulário de entrada à esquerda e visualização do QR gerado + botão de download à direita. Ideal para iniciantes aprenderem estruturação, estilização e integração de bibliotecas externas.
 
# Gerador de Código QR

Projeto simples e funcional para gerar códigos QR de textos, links ou mensagens, com interface moderna e responsiva.

## 🚀 Funcionalidades
- Entrada de texto ou URL
- Geração instantânea do código QR
- Visualização ao lado do formulário
- Botão para baixar a imagem em PNG
- Suporte a tecla Enter para gerar
- Layout adaptável para celulares e telas grandes

## 🛠️ Tecnologias
- *HTML5*: estrutura da página
- *CSS3*: estilos, flexbox e responsividade
- *JavaScript*: lógica de geração e interação

## 📁 Estrutura
 
 
projeto-qr/
├── index.html   # Página principal
├── style.css    # Estilos visuais
└── script.js    # Regras e funções
 
plaintext
  

## ▶️ Como usar
1. Baixe ou clone os arquivos em uma pasta
2. Abra index.html no navegador
3. Digite o conteúdo no campo de texto
4. Clique em *Gerar Código QR* ou aperte Enter
5. O resultado aparece à direita; clique em *Baixar QR* para salvar
